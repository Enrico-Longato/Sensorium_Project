# Sensorium_Project

This repository is the result of the project of Aplied AI in the frame of the Master in Data Management and Curation (2025-2026 edition). 
The project is focused on the Sensorium 2023 competition (https://www.sensorium-competition.net/) and the goals are:
- Label videos by type: Natural Movies, Natuale Images, Directional Pink Noise, Random Dot, Gaussian Dot, Drifting Gabor
- Build a visualization tool for the exploration of the whole dataset
